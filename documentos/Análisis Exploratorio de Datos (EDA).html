Guía de Preguntas para Análisis Exploratorio de Datos (EDA)

Estas preguntas te guiarán en la exploración inicial de tu conjunto de datos para descubrir patrones, anomalías y relaciones significativas

📊 Comprensión General del Dataset

  1. ¿Cuál es la estructura del dataset? (número de filas y columnas)
    Objetivo: Entender el tamaño y alcance de los datos
  2. ¿Qué tipos de datos contiene cada columna? (numéricos, categóricos, fechas, etc.)
    Objetivo: Identificar el formato de cada variable
  3. ¿Existen valores faltantes en el dataset? ¿En qué columnas y en qué proporción?
    Objetivo: Evaluar la completitud de los datos
  4. ¿Hay duplicados en el dataset? ¿Cuántos y dónde se encuentran?
    Objetivo: Identificar posibles problemas de calidad de datos
  5. ¿Cuál es el rango temporal de los datos? (si aplica)
    Objetivo: Comprender el período cubierto por los datos

📈 Análisis de Variables Numéricas

  1. ¿Cuáles son las distribuciones de las variables numéricas? (histogramas, boxplots)
    Objetivo: Identificar formas de distribución, sesgos y valores atípicos
  2. ¿Cuáles son los estadísticos descriptivos clave? (media, mediana, desviación estándar, rango)
    Objetivo: Resumir las características centrales de los datos
  3. ¿Existen valores atípicos significativos? ¿Cómo afectan al análisis?
    Objetivo: Identificar anomalías que puedan distorsionar resultados
  4. ¿Cómo se relacionan las variables numéricas entre sí? (matriz de correlación)
    Objetivo: Descubrir relaciones lineales entre variables
  5. ¿Cuál es la variabilidad de las variables clave? ¿Son estables o muestran dispersión?
    Objetivo: Evaluar la consistencia de los datos

📋 Análisis de Variables Categóricas

  1. ¿Cuáles son las categorías principales y sus frecuencias?
    Objetivo: Identificar categorías dominantes y minoritarias
  2. ¿Existen categorías desbalanceadas que puedan afectar el análisis?
    Objetivo: Detectar posibles sesgos en la distribución
  3. ¿Cómo se relacionan las variables categóricas con las numéricas? (boxplots por categoría)
    Objetivo: Explorar diferencias entre grupos
  4. ¿Existen categorías poco representadas que deban agruparse?
    Objetivo: Simplificar categorías con pocas observaciones
  5. ¿Hay relaciones entre diferentes variables categóricas? (tablas cruzadas)
    Objetivo: Identificar patrones de asociación

🔍 Relaciones y Patrones

  1. ¿Existen tendencias temporales en los datos? (si hay variables de tiempo)
    Objetivo: Identificar patrones a lo largo del tiempo
  2. ¿Cómo se relacionan las variables predictoras con la variable objetivo?
    Objetivo: Explorar relaciones clave para modelado
  3. ¿Existen agrupaciones naturales en los datos? (análisis de clusters)
    Objetivo: Descubrir segmentos ocultos
  4. ¿Hay patrones estacionales o cíclicos en los datos?
    Objetivo: Identificar repeticiones a intervalos regulares
  5. ¿Existen relaciones no lineales que no se capturan con correlaciones?
    Objetivo: Detectar patrones complejos

⚠️ Calidad de Datos y Transformaciones

  1. ¿Cómo manejar los valores faltantes? (eliminación, imputación, etc.)
    Objetivo: Determinar estrategia para datos incompletos
  2. ¿Necesitan normalización o estandarización las variables numéricas?
    Objetivo: Preparar datos para algoritmos sensibles a escalas
  3. ¿Es necesario codificar variables categóricas? (one-hot, label encoding)
    Objetivo: Preparar datos para algoritmos que requieren entradas numéricas
  4. ¿Se requieren transformaciones de variables? (log, raíz cuadrada, binning)
    Objetivo: Mejorar linealidad o manejar sesgos
  5. ¿Hay inconsistencias en los datos que requieran limpieza? (errores de entrada, formatos inconsistentes)
    Objetivo: Asegurar consistencia en todo el dataset

Estas preguntas son un punto de partida - adapta según las características específicas de tu dataset
